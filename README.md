# Modi Reviews Sentiment Analysis

## 📌 Project Overview

This project focuses on performing **Sentiment Analysis on Modi reviews** using Natural Language Processing (NLP) and Machine Learning techniques. The system analyzes text reviews and classifies them into different sentiment categories based on the opinions expressed in the reviews.

The project includes data preprocessing, feature extraction using TF-IDF, model training, and deployment using Streamlit.

---

## 🚀 Features

- Text data cleaning and preprocessing
- Removal of unwanted characters and stopwords
- Text vectorization using TF-IDF
- Machine Learning-based sentiment classification
- Saved trained model and vectorizer using Pickle
- Interactive Streamlit web application for predictions

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- TF-IDF Vectorizer
- Streamlit
- Pickle
